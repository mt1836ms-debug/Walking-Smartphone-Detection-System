# Walking-Smartphone-Detection-System
# 歩きスマホ検知システム

## 概要
スマートフォンの加速度・角速度などのセンサーデータを用いて、歩きスマホをリアルタイムで検知する機械学習システムです。

大学の認知情報科学実験で6人チームで開発を行い、私は主に**センサーデータの前処理・特徴量選択・機械学習モデルの構築・精度改善**を担当しました。

## システム概要

- センサーデータの取得
- 前処理・特徴量生成
- 機械学習モデルによる歩きスマホ判定
- リアルタイム検知
- モデル評価
- ログイン認証
- メール通知
- データベースへの設定保存

---

## 担当した内容

### センサーデータの前処理
- 欠損値処理
- データ型変換
- 特徴量の選定
- 学習データと推論データの前処理を統一

### 機械学習モデルの改善
- Random Forestによる分類モデルを構築
- 特徴量重要度を分析し不要な特徴量を削除
- パラメータ調整による精度改善
- 学習データだけでなく未知データでも安定して判定できるよう改善

### リアルタイム推論
- 学習済みモデルを用いたリアルタイム判定処理を実装

## 成果

- センサーデータから歩きスマホを自動判定するシステムを実装
- 前処理・特徴量選択を改善し、高い分類精度を実現
- GUIから学習・評価・リアルタイム検知まで操作可能
- ログイン・メール通知・設定保存機能を実装し、実際の利用を意識したシステムを構築

---

## 工夫した点

### データ前処理
学習データとリアルタイムデータで同じ前処理になるよう設計し、推論時のエラーを防止しました。

### 特徴量選択
特徴量重要度を分析し、不要な特徴量を削除することで、モデルの汎化性能向上を図りました。

### 利用者を意識した設計
専門知識がなくても利用できるよう、GUIによるモデル作成・評価・リアルタイム検知を実装しました。
