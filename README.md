# Walking Smartphone Detection System

歩きスマホをリアルタイムで検知し、必要に応じてメール通知を行う機械学習Webアプリケーションです。

大学の認知情報科学実験における6人チームでの開発プロジェクトで、私は**センサーデータの前処理・特徴量選択・機械学習モデルの改善・リアルタイム処理**を担当しました。

---

## 概要

スマートフォンの加速度・角速度センサーデータを利用し、歩きスマホをリアルタイムで判定するシステムです。

収集したセンサーデータを前処理し、Random Forestによる機械学習モデルを構築しました。また、Streamlitを用いてモデル学習・評価・リアルタイム判定を行えるWebアプリケーションとして実装しています。

---

## 工夫した点

### データ前処理

学習データとリアルタイムデータで同じ前処理を適用し、推論時のエラーを防止しました。また、不要なデータを除去することで学習データの品質向上を図りました。

### 特徴量設計

平均・最大・最小値に加え、四分位数や歪度、尖度などの統計量を特徴量として追加し、モデルの識別性能と汎化性能を向上させました。

### 利用者を意識した設計

GUIによる操作を採用し、専門知識がなくてもモデル学習・評価・リアルタイム検知が行えるよう設計しました。また、メール通知や設定保存機能を追加し、継続利用しやすいシステムを目指しました。

## 使い方

### 1. ログイン

アプリを起動し、ログイン後に各機能を利用できます。

### 2. モデル学習

- 訓練データ（Excel）
- テストデータ（Excel）

をアップロードすると学習済みモデル（`.pkl`）が作成・保存されます。

### 3. モデル評価

保存したモデルを読み込み、以下の指標を表示します。

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix

### 4. リアルタイム検知

1. スマートフォンでPhyphoxを起動
2. Remote Accessを有効化
3. PCと同じWi-Fiへ接続
4. IPアドレスを入力
5. 「開始」をクリック

リアルタイムで取得したセンサーデータから歩きスマホを判定します。

### 5. メール通知

歩きスマホを検知すると登録したメールアドレスへ通知できます。

通知方法

- 検知時に即時送信
- 計測終了後にまとめて送信

### 6. 設定保存

モデルや通知設定などはデータベースへ保存され、次回起動時にも利用できます。
